About AskNova

AskNova is a modern search engine that helps you find information quickly and efficiently.

Documentation

How to Use AskNova

Simply enter your search query and click Search or press Enter.

Advanced Search Features

  • Voice Search: Click the microphone icon to search by voice
  • Search Filters: Filter results by category, type, date, and rating
  • Search Modes: Switch between Web, Images, Videos, and News search
  • Keyboard Navigation: Use arrow keys to navigate results, Enter to open
  • Favorites: Save important results for later reference

Search Tips

  • Use specific keywords for better results
  • Try different combinations of words
  • Use quotes for exact phrases
  • Filter by category to narrow down results
  • Sort results by date, rating, or relevance

Keyboard Shortcuts

  • Ctrl/Cmd + K: Focus search box
  • Escape: Return to home page
  • Arrow Keys: Navigate search results
  • Enter: Open selected result
